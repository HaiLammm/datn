# Sample Job Descriptions

## Sample 1: Backend Developer - Middle Level

### Công ty: TechViet Solutions
### Vị trí: Backend Developer
### Cấp độ: Middle (2-4 năm kinh nghiệm)

**Mô tả công việc:**
- Phát triển và maintain RESTful APIs cho các ứng dụng web/mobile
- Thiết kế database schema và optimize queries
- Tích hợp với các third-party services (payment, SMS, email)
- Code review và hỗ trợ junior developers
- Viết unit tests và integration tests
- Tham gia sprint planning và technical discussions

**Yêu cầu bắt buộc:**
- 2-4 năm kinh nghiệm với Python
- Thành thạo ít nhất 1 framework: FastAPI, Django, hoặc Flask
- Kinh nghiệm với PostgreSQL hoặc MySQL
- Hiểu biết về RESTful API design principles
- Kinh nghiệm với Git và Git workflow
- Biết về authentication/authorization (JWT, OAuth2)
- Kỹ năng giao tiếp tốt, làm việc nhóm

**Yêu cầu ưu tiên:**
- Có kinh nghiệm với Docker và Docker Compose
- Biết về CI/CD (GitLab CI, GitHub Actions, Jenkins)
- Kinh nghiệm với Redis, Celery
- Hiểu về Microservices architecture
- Có kinh nghiệm với AWS/GCP/Azure
- Biết về message queues (RabbitMQ, Kafka)

**Quyền lợi:**
- Lương: 20-30 triệu (tùy năng lực)
- Thưởng theo performance
- Làm việc hybrid (2-3 ngày/tuần WFH)
- Đào tạo công nghệ mới
- Team building, du lịch hàng năm

## Sample 2: Senior Backend Engineer

### Công ty: FinTech Innovations
### Vị trí: Senior Backend Engineer
### Cấp độ: Senior (4+ năm kinh nghiệm)

**Mô tả công việc:**
- Thiết kế và phát triển high-performance, scalable backend systems
- Lead technical design và architecture decisions
- Mentor junior/middle developers
- Optimize system performance và troubleshoot production issues
- Implement best practices cho security, testing, monitoring
- Collaborate với Product, DevOps, Frontend teams

**Yêu cầu bắt buộc:**
- 4+ năm kinh nghiệm backend development
- Expert level với Python và async programming
- Deep understanding của database design và optimization
- Kinh nghiệm với distributed systems và microservices
- Strong understanding về system design và scalability
- Kinh nghiệm với monitoring tools (Prometheus, Grafana)
- Excellent problem-solving và debugging skills

**Yêu cầu ưu tiên:**
- Kinh nghiệm trong fintech hoặc payment domain
- Biết về blockchain/cryptocurrency
- Có kinh nghiệm lead team hoặc mentor
- Contributions to open source projects
- Certifications (AWS, GCP, etc.)

**Quyền lợi:**
- Lương: 35-50 triệu + performance bonus
- Stock options
- Full remote hoặc hybrid
- Budget cho learning & development
- Top-tier hardware và tools

## Sample 3: Junior Python Developer

### Công ty: StartUp EdTech
### Vị trí: Junior Python Developer
### Cấp độ: Junior (0-2 năm kinh nghiệm)

**Mô tả công việc:**
- Phát triển features mới cho platform giáo dục online
- Fix bugs và maintain existing codebase
- Viết unit tests cho code
- Tham gia code review và học hỏi từ senior members
- Document code và technical processes

**Yêu cầu bắt buộc:**
- 0-2 năm kinh nghiệm với Python (hoặc fresh graduate có project)
- Hiểu biết cơ bản về web development
- Biết SQL cơ bản
- Có kinh nghiệm với Git
- Eager to learn và có thái độ tích cực
- Good communication skills

**Yêu cầu ưu tiên:**
- Đã làm project với Django hoặc FastAPI
- Biết về HTML/CSS/JavaScript cơ bản
- Có personal projects hoặc GitHub portfolio
- Hiểu về Agile/Scrum

**Quyền lợi:**
- Lương: 10-15 triệu
- Mentorship từ senior developers
- Cơ hội học hỏi và phát triển
- Môi trường startup năng động
- Snacks, coffee miễn phí
